# Drop your raw materials here

- `resume.pdf` — your résumé (PDF). Primary source for onboarding.
- `links.txt` — one URL per line (your website, your LinkedIn).
- `work-samples/` — case-study source material (`*.md`, `*.pdf`, `*.txt`).

Then run `./onboard.sh` from the project root to generate `profile.yaml` and
`case-studies/` (you review/edit before first use).
